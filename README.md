# Student Score Analyzer

## Mô tả
Ứng dụng Java phân tích điểm số học sinh.

### Chức năng
- Đếm số học sinh giỏi (>= 8.0)
- Tính điểm trung bình hợp lệ (0-10)

## Cấu trúc thư mục
- `src/` - chứa mã nguồn Java
- `test/` - chứa unit test với JUnit
